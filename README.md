## Sprints

### Sprint 1 — Développement Web

* **Période** : Semaine 1 et Semaine 3
* **Durée totale** : 10 jours ouvrés

Ce sprint est scindé en deux parties :

* **Partie Public (S1)** : développement de l’interface utilisateur destinée aux utilisateurs finaux

* **Partie Admin (S3)** : développement de l’interface d’administration et des fonctionnalités de gestion

* **Avancement** :
  Application mobile : terminée et validée
  Partie Web Public : en cours
  Partie Web Admin : planifiée après la période de vacances
