# Planification du Projet

---
## Lab : NativePHP

### Objectif

Développer une **application mobile hybride performante** en utilisant **Laravel** et **NativePHP**, avec une intégration API fluide et optimisée.

### Phases du projet

1. **Installation et Configuration**  
   Mise en place de l’environnement et création d’un projet Hello World avec NativePHP.

2. **Consommation d’API**  
   Connexion et gestion des données via une API (affichage).

3. **Personnalisation de l’application**  
   Modification de l’icône, du splash screen et des éléments visuels de l’application.
